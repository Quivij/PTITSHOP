# Phân chia màn hình Mobile - Nhóm 4 người

> Lưu ý: Làm giao diện **Mobile**, không phải Web.

## Tổng danh sách màn

1. **Home** (tính 2 màn: Home chính + màn Search / Products)
2. **Chi tiết sản phẩm**
3. **Login + Register** (+ Forgot Password + Verify OTP)
4. **Giỏ hàng**
5. **Preview Order + Thanh toán thành công** (Checkout + Payment callback)
6. **Danh sách đơn hàng** (Orders)
7. **Profile** (Edit profile + Địa chỉ + Sản phẩm yêu thích)

---

## Chia 4 phần (4 người)

### Phần 1: Khám phá sản phẩm
- **Home**
- **Search / Danh sách sản phẩm**
- **Danh mục** (Category page)

### Phần 2: Xác thực
- **Login**
- **Register**
- **Forgot Password**
- **Verify OTP**

### Phần 3: Mua hàng
- **Giỏ hàng**
- **Checkout** (Preview Order + chọn địa chỉ + chọn phương thức thanh toán)
- **Thanh toán thành công** (Modal kết quả / Payment callback)

### Phần 4: Quản lý
- **Danh sách đơn hàng**
- **Profile** (Thông tin cá nhân + Edit profile)
- **Địa chỉ** (CRUD địa chỉ nhận hàng)
- **Sản phẩm yêu thích**
